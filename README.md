# FizzBuzz

Simple **FizzBuzz** application with OOP and SOLID principles. Also it uses several of *GoF* patterns

## Solution structure

- App - simple console application for ***demonstration purpose only***
- Common - main library with FizzBuzz logic
- Tests - unit-tests

## Stack

- DotNet 7
- MSTest
- Moq

## Patterns

- Chain of responsibility
- Factory
- Strategy
- Builder
- Decorator
- Specification
